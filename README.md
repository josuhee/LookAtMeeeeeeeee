### ✔ Install
Clone repo and install requirements.txt in a Python>=3.7.0 environment, including PyTorch>=1.7.
```
git clone https://github.com/josuhee/LookAtMeeeeeeeee  # clone
cd LookAtMeeeeeeeee
pip install -r requirements.txt  # install
```

### ✔ Run
```
cd src
python main.py
```
